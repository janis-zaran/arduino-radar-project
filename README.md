# Arduino Radar Project

Radar project with:

- Arduino UNO Q
- Servo motor
- Ultrasonic sensor
- Passive buzzer
- Display (planned)

## Project structure

- `tests/` for component tests  
- `src/` for final project code  
- `wiring/` for connection notes and diagrams  
- `docs/` for planning and notes  
- `media/` for photos and videos
